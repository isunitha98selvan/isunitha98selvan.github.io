Selected Course Projects

Predicting ICU Mortality Using Vital Signs and Clinical Notes [Report]

  • We predict the Mortality rate for an ICU in a span of 48 hours for patients > 18 years of age using lab events, chart events and patient data from the MIMIC-III dataset.
  • We perform various data analytics techniques to process the data from multiple tables and generate a time series data for 48 hours for each patient admission. The computed feature vectors are passed through various classification models and an LSTM to predict the mortality.

    • Emotion Recognition and Behavioral Analysis of Multimodal Data [Slides]

  • We built a web application which can be used to train candidates for interviews. The system analyses three modalities - video, audio and any text written by the candidate. It generates an emotion report for the entire duration of the interview.
  • We use CNN and RNN to capture the spatial and temporal features of each modality. All the modalities are combined using Attention and a softmax layer.

    • End to End Secure Chat Client [Report]

  • Worked on implementing the Double Ratchet algorithm to ensure perfect forward secrecy as part of a simple messaging flask application.
  • Various improvements were made for recovery from compromise and secure deletion of plaintext.

    • Steganography combining LSB and PVD in a block [Report]

  • Worked on combining both LSB Substitution and Pixel Value Differencing method to decrease the probability of detection by any Steg expose tools.
  • We achieved good undetectability capacity but with lesser hiding capacity. One cover pixel is hidden within 2x2 dimensional carrier pixelblock.
