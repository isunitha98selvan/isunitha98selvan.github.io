Medical Question Answering System

The project was done as part of the MediQA 2019 challenge.
1) NLI: This first task consists in identifying three inference relations between two sentences: Entailment, Neutral and Contradiction.
2) RQE: This task focuses on identifying entailment between two questions in the context of QA.
3) QA: The objective of this task is to filter and improve the ranking of automatically retrieved answers. The input ranks are generated by the medical QA system CHiQA.
Link to the Github repository: ACL-BioNLP-2019 Shared Task
